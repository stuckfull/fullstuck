# FullStuck.php (v0.3.0)

FullStuck.php adalah framework mikro yang dirancang untuk kecepatan pengembangan maksimal tanpa mengorbankan fitur modern. Seluruh core framework berada dalam **satu file tunggal**, membuatnya sangat mudah di-deploy ke shared hosting mana pun tanpa Composer. Pada versi 0.3.0, FullStuck berevolusi menjadi Micro-Framework lincah dengan kapabilitas *Hybrid Front-End Routing* (FST Agent).
